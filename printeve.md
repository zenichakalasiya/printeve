Business idea : our platform connects the user and printer online. but user does not know which printer prints his print. user can just upload their file on our website and fill some necessary details regarding printing. user can create multiple products for example, user want a print of sticker and some paper document. so user can first select the category of sticker -> upload details and files -> save (saved products can store in cart) -> select category of paper -> upload file and details -> save -> cart -> payment -> order tracking screen.
