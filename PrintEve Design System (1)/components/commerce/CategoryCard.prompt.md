**CategoryCard** — product-category tile with a brand illustration, name and description; lifts on hover.

```jsx
<CategoryCard name="Business Cards"
  description="Premium finishes that make a strong first impression"
  iconSrc="assets/illustrations/business-cards.svg"
  href="/products?category=business-cards" />
```

Pass `iconSrc` (illustration URL) or an inline `icon` node.
