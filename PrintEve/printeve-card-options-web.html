Web / responsive layouts
The same three cards, same components, same palettes — re-flowed for a desktop viewport. In every version the profile is pinned to a left rail (it stays put while you scroll), every other section stretches across the full width, and the stacked vertical lists (contact, services, details, files) lay out horizontally. Layout is driven by container queries, so each card reflows on its own width — drag the bottom-right corner of any window below ~900px and it snaps back to the phone layout.
